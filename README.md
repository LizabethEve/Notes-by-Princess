# Notes-by-Princess

# Description
  A notes app that started as a fun CustomTkinter project quickly turned into a full on notes editor with big goals. The first priority is to get Notes by Princess to a spot where I would use it as my daily notes app, and possibly get it converted to a mobile app.

# USE
  Code is free to use and copy for personal use, this is a hobby project and I want others to have fun as well:) UNLESS you're a business/company/organization, in which case you are not permitted to use or copy any code from this project

# Realistic Goals
  - Fix key capture so letters aren't missed when applying tags
    
  - Add custom images to personalize
    
  - Add an automatic update check to check for new versions or updates on the project          github and install in place of the current version before the main application opens
    
  - Add font repository as well as button functionality including buttons in the bottom        frame, and a size option
    
  - Add custom theme option where user can pick any color from a color wheel or hex map,       and assign it to a specific widget type (widget color types at the top of notes.py)

  - Add an underline button

  - Add a cross through button

  - Add a print button

  - Add a share button where user can choose to export file(s): as json and open it on         another device with Notes by Princess, as a plain text .txt document, and as a pdf

  - Add a split screen option to view multiple files at a time

# Unrealistic Goals
